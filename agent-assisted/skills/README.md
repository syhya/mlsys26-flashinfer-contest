# Skills

This directory vendors the agent workflow skills used during optimization and
submission preparation.

| Skill | Primary use |
| --- | --- |
| `flashinfer-b200-contest-optimizer` | MLSys 2026 FlashInfer B200 optimization loop with Modal benchmarking and NCU analysis. |
| `flashinfer-submission-tagger` | Submission tag creation and config topology checks. |

The top-level repository README includes a reusable prompt template for
continuing optimization with multiple sub-agents.
