## Phase 1: Mathematical Definition

For each `(t, h)`:
- `q = concat(q_nope[t,h], q_pe[t,h]) ∈ R^576`
- For `j ∈ [0, 2048)`: `raw = sparse_indices[t,j]`. If `raw == -1`, neutral. Else `v_j = ckv_cache[raw/64, raw%64, :]`, `kpe_j = kpe_cache[raw/64, raw%64, :]`, `z_j = sm_scale * (<q_nope, v_j> + <q_pe, kpe_j>)`.
- `out[t,h] = Σ softmax(z)_j v_j`, `lse[t,h] = log2(Σ exp(z_j))`.

**Online softmax state** `(m, l, O ∈ R^512)`. Neutral = `(-INF, 0, 0)`.

**Partitioning:** Split 2048 slots into S=64 partitions of 32 slots each. Each partition produces `(m_p, l_p, O_p)`. Exact merge:
- `m = max_p m_p`
- `scale_p = exp(m_p - m) if m_p ≠ -INF else 0`
- `l = Σ l_p * scale_p`, `O = Σ O_p * scale_p`
- `out = O/l if l > 0 else 0`; `lse = m·log2(e) + log2(l) if l>0 else -INF`

**Variables X (the structural change):**
- Compute grid: `(S=64, T, H=16)` instead of `(S=64, T)`.
- Compute block: `(32, 1)` = 32 threads (1 warp) instead of `(32, 16)` = 512 threads.
- Each block now handles exactly ONE `(t, s, h)` tuple.
- At T=1: grid = 64*1*16 = **1024 blocks** vs previous 64 blocks (16x boost).

**Constraints C:** Exact `-1` masking; FP32 accumulation; BF16 store; base-2 lse; explicit BF16 intrinsics; preserve ABI + `PYBIND11_MODULE`.

### Step 2: Rewrite kernel body — single-warp, single-head
```cpp
__global__ void split_k_compute_kernel(...) {
    int s = blockIdx.x;          // partition [0, 64)
    int t = blockIdx.y;          // token
    int h = blockIdx.z;          // head [0, 16)
    int lane = threadIdx.x;      // [0, 32)

    __shared__ alignas(16) int idx_shared[32];
    __shared__ alignas(16) __nv_bfloat16 smem_Kc[32 * 512];   // 32KB
    __shared__ alignas(16) __nv_bfloat16 smem_Kp[32 * 64];    // 4KB
    // Total smem ≈ 36KB — fits comfortably.

    // ---- Load q for this single head into registers ----
    float2 q_n_f32[8];
    #pragma unroll
    for (int j = 0; j < 8; ++j) {
        __nv_bfloat162 q_val = *reinterpret_cast<const __nv_bfloat162*>(
            &q_nope_ptr[t * 16 * 512 + h * 512 + j * 64 + lane * 2]);
        q_n_f32[j] = __bfloat1622float2(q_val);
    }
    __nv_bfloat162 q_pe_v = *reinterpret_cast<const __nv_bfloat162*>(
        &q_pe_ptr[t * 16 * 64 + h * 64 + lane * 2]);
    float2 q_p_f32 = __bfloat1622float2(q_pe_v);

    float2 O_reg[8];
    #pragma unroll
    for (int j = 0; j < 8; ++j) O_reg[j] = make_float2(0.0f, 0.0f);
    float m = -INFINITY;
    float l = 0.0f;

    // ---- Load 32 sparse indices (one warp covers all 32) ----
    if (lane < 32) idx_shared[lane] = sparse_indices_ptr[t * 2048 + s * 32 + lane];
    __syncwarp();

    // ---- Stage 32 keys' Kc into smem ----
    // 32 keys × 512 bf16 = 16384 bf16 = 1024 float4. Per warp = 1024/32 = 32 float4 per lane.
    // Layout per lane: lane handles loads strided by 32.
    #pragma unroll
    for (int i = 0; i < 32; ++i) {
        int idx = idx_shared[i];
        if (idx != -1) {
            // 32 lanes × float4 (8 bf16) = 256 bf16 per pass; need 2 passes per row
            const float4* src = reinterpret_cast<const float4*>(&ckv_cache_ptr[idx * 512]);
            float4* dst = reinterpret_cast<float4*>(&smem_Kc[i * 512]);
            // 512 bf16 = 64 float4; 32 lanes × 2 = 64
            dst[lane]      = src[lane];
            dst[lane + 32] = src[lane + 32];
        }
    }
    // ---- Stage Kp: 32 keys × 64 bf16 = 2048 bf16 = 128 float4. 32 lanes × 4 = 128 ----
    #pragma unroll
    for (int i = 0; i < 32; ++i) {
        int idx = idx_shared[i];
        if (idx != -1) {
            const float4* src = reinterpret_cast<const float4*>(&kpe_cache_ptr[idx * 64]);
            float4* dst = reinterpret_cast<float4*>(&smem_Kp[i * 64]);
            // 64 bf16 = 8 float4; only need 8 lanes
            if (lane < 8) dst[lane] = src[lane];
        }
    }
    __syncwarp();

    // ---- Online softmax over 32 keys (UNCHANGED math from iter20) ----
    #pragma unroll
    for (int i = 0; i < 32; ++i) {
        if (idx_shared[i] == -1) continue;
        float local_dot = 0.0f;
        float2 k_f_reg[8];
        #pragma unroll
        for (int j = 0; j < 8; ++j) {
            __nv_bfloat162 k_n = *reinterpret_cast<const __nv_bfloat162*>(
                &smem_Kc[i * 512 + j * 64 + lane * 2]);
            float2 k_f = __bfloat1622float2(k_n);
            k_f_reg[j] = k_f;
            local_dot = fmaf(q_n_f32[j].x, k_f.x, local_dot);
            local_dot = fmaf(q_n_f32[j].y, k_f.y, local_dot);
        }
        __nv_bfloat162 k_p = *reinterpret_cast<const __nv_bfloat162*>(
            &smem_Kp[i * 64 + lane * 2]);
        float2 k_pf = __bfloat1622float2(k_p);
        local_dot = fmaf(q_p_f32.x, k_pf.x, local_dot);
        local_dot = fmaf(q_p_f32.y, k_pf.y, local_dot);

        // Warp reduction (single warp, no need for syncwarp before shfl)
        #pragma unroll
        for (int off = 16; off > 0; off /= 2)
            local_dot += __shfl_down_sync(0xffffffff, local_dot, off);
        float logit = __shfl_sync(0xffffffff, local_dot, 0) * sm_scale;

        float m_new = fmaxf(m, logit);
        float exp_diff = (m == -INFINITY) ? 0.0f : __expf(m - m_new);
        float exp_logit = __expf(logit - m_new);
        l = fmaf(l, exp_diff, exp_logit);
        m = m_new;
        #pragma unroll
        for (int j = 0; j < 8; ++j) {
            O_reg[j].x = fmaf(exp_logit, k_f_reg[j].x, O_reg[j].x * exp_diff);
            O_reg[j].y = fmaf(exp_logit, k_f_reg[j].y, O_reg[j].y * exp_diff);
        }
    }

    // ---- Store partial state. Layout UNCHANGED: O_tmp[t, s, h, 512] ----
    #pragma unroll
    for (int j = 0; j < 8; ++j) {
        int base_idx = t * 64 * 16 * 512 + s * 16 * 512 + h * 512 + j * 64 + lane * 2;
        *reinterpret_cast<float2*>(&O_tmp[base_idx]) = make_float2(O_reg[j].x, O_reg[j].y);
    }
    if (lane == 0) {
        m_tmp[t * 16 * 64 + h * 64 + s] = m;
        l_tmp[t * 16 * 64 + h * 64 + s] = l;
    }
}
```

## Phase 3: Safety Valve

1. **Masking:** `if (idx_shared[i] == -1) continue;` and `if (idx != -1)` guards on all cache loads — preserved from iter20.
2. **Empty partition:** If all 32 sparse slots in (t,s) are -1, kernel writes m=-INF, l=0, O=0 → reducer treats as neutral (already handled by `if (m_s_smem[i] != -INFINITY)` and `coeff > 0` checks).
3. **Empty row:** Reducer writes 0 output and -INF lse if `m_max == -INFINITY || l_sum <= 0`.
4. **Mathematical equivalence:** Each `(t, s, h)` computation is independent — no cross-thread dependence on the head axis existed before either (the previous block's 16 warps each computed a different head independently). This refactor is just relabeling `threadIdx.y` → `blockIdx.z`. **Bitwise-identical FP32 results guaranteed.**
5. **Smem fits:** ~36KB per block, well under 48KB default limit. 32-thread blocks → very high theoretical occupancy.
6. **Sync simplification:** Single warp = no `__syncthreads()` needed; use `__syncwarp()` (or none, since lane writes are read by same warp after the implicit warp-coherent ordering of shared memory writes within a warp — but `__syncwarp()` after the smem stores is safest).
7. **ABI:** `dsa_forward` signature + `PYBIND11_MODULE` preserved exactly.
8. **Fallback:** If anything goes wrong, the large path (num_tokens >= 128) is untouched; correctness can only be impacted on small-T which is the only regime we're modifying.
