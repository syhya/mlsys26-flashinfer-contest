- Constraints C: exact online-softmax merge semantics; FP32 accumulation; BF16 output; base-2 lse.
- Objective f(X): minimize reducer latency while preserving correctness 1.0.

Advantages: Tiny delta from proven winner; preserves all correctness guarantees; removes real overhead.
