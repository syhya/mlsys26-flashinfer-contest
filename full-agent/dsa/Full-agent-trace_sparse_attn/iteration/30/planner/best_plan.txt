## Phase 1: Mathematical Definition (UNCHANGED from parent)

For each token `t`, head `h`:
- Logit: `z_j = sm_scale * (q_nope[t,h] · k_nope(idx_j) + q_pe[t,h] · k_pe(idx_j))` for valid `idx_j != -1`.
- Online softmax state per head: `(m, l, O)` in FP32.
- Update for one valid key (logit `z`, value `v`):
  - `m' = max(m, z)`
  - `α = (m == -INF) ? 0 : exp(m - m')`
  - `β = exp(z - m')`
  - `l' = l*α + β`
  - `O' = O*α + β*v`
- Final: `out = O/l`, `lse = m * log2(e) + log2(l)` if `l>0` else `-INF`.
- Merge of two partial states A,B: `m=max(m_A,m_B); s_A=exp(m_A-m); s_B=exp(m_B-m); l = l_A s_A + l_B s_B; O = O_A s_A + O_B s_B`.

These equations are *invariant* across the proposed refactor.

## Phase 2: The Optimization Loop (Deterministic Refinement Application)

This is a finite list of *byte-level deltas* to apply to the parent. No randomness, no multi-start needed.

### Delta D1 — Fused-path tile early-skip via separate flag
In `dsa_forward_kernel` (the large path, used when `num_tokens >= 128`):
- Add a shared variable `__shared__ int tile_has_valid;`
- After loading `idx_shared[32]` and syncing, let `tid==0` compute:
  ```
  int any_valid = 0;
  #pragma unroll
  for (int i=0;i<32;++i) any_valid |= (idx_shared[i] != -1);
  tile_has_valid = any_valid;
  ```
- After `__syncthreads()`, `if (!tile_has_valid) { __syncthreads(); continue; }`
- This skips both BF16 K loads and 32 inner iterations for fully-padded tiles, which is common for short sequences.
- Add `#pragma unroll 1` on the outer `for (int tile=0; tile<64; ++tile)` to control register pressure.

### Delta D2 — Reducer simplification (eliminate `l_s_smem`)
In `split_k_reduce_kernel`:
- **Remove** `__shared__ float l_s_smem[64];`
- Lane-0 loop becomes:
  ```

  for (int i=0;i<64;++i) m_max = fmaxf(m_max, m_s_smem[i]);
  float l_sum = 0.0f;
  for (int i=0;i<64;++i) {
      if (m_s_smem[i] != -INFINITY) {
          float scale = __expf(m_s_smem[i] - m_max);
          scale_smem[i] = scale;
          l_sum = fmaf(l_tmp[t*16*64 + h*64 + i], scale, l_sum);
      } else {
          scale_smem[i] = 0.0f;
      }
  }
  m_global = m_max; l_global = l_sum;
  lse_ptr[t*16+h] = (m_max == -INFINITY || l_sum <= 0.0f) ? -INFINITY

  ```
  Note: `scale_smem[i]` here holds `exp(m_i - m_global)`, NOT yet divided by `l_global`.

## Phase 3: The "Safety Valve" (CRITICAL — Correctness Guarantee)

### S1 — Mask invalidation
Every `idx == -1` is masked at three levels:
1. **Load level:** `if (idx != -1) { float4 load... }` — already present in both load loops (CKV and KPE).
2. **Compute level:** `if (idx_shared[i] == -1) continue;` inside the 32-key inner loop — already present.
3. **Tile level (NEW via D1):** entire 32-key tile skipped if all -1, and online state remains unchanged (neutral).

### S2 — Empty-row detection in reducer
The expression `(m_max == -INFINITY || l_sum <= 0.0f) ? -INFINITY : ...` (D2) explicitly handles the empty-row case. Output uses `if (m_global != -INFINITY && l_global > 0.0f) { ... }` guard, leaving the four float accumulators at 0. Result: zero output, `-INF` lse.

### S3 — FP32 throughout
All `m`, `l`, `O_reg`, dot products, `scale_smem`, `inv_l` are float32. BF16 is used ONLY at:
- Input loads (`__bfloat1622float2`)
- Final output store (`__floats2bfloat162_rn`)

### S4 — Build environment compliance
- Keep `#include <torch/extension.h>` and `#include <cuda_bf16.h>`.
- Use `__bfloat1622float2` / `__floats2bfloat162_rn` exclusively.
- No implicit BF16↔float casts.
- Preserve `PYBIND11_MODULE(TORCH_EXTENSION_NAME, m) { m.def("dsa_forward", &dsa_forward, "DSA Forward"); }`.

### S5 — Fallback
If any new branch causes compile error, the executor must fall back to the parent verbatim plus only D2+D3 (reducer changes), since those are the highest-leverage and lowest-risk.

__global__ void split_k_reduce_kernel(
    const float* __restrict__ O_tmp,
    const float* __restrict__ m_tmp,
    const float* __restrict__ l_tmp,
    __nv_bfloat16* __restrict__ output_ptr,
    float* __restrict__ lse_ptr) {
    int t = blockIdx.x; int h = blockIdx.y; int lane = threadIdx.x;
    __shared__ float m_global, l_global;
    __shared__ float m_s_smem[64];
    __shared__ float scale_smem[64];

    if (lane < 64) m_s_smem[lane] = m_tmp[t*16*64 + h*64 + lane];
    __syncthreads();

    if (lane == 0) {
        float m_max = -INFINITY;
        #pragma unroll
        for (int i=0;i<64;++i) m_max = fmaxf(m_max, m_s_smem[i]);
        float l_sum = 0.0f;
        #pragma unroll
        for (int i=0;i<64;++i) {
            if (m_s_smem[i] != -INFINITY) {
                float sc = __expf(m_s_smem[i] - m_max);
                scale_smem[i] = sc;
                l_sum = fmaf(l_tmp[t*16*64 + h*64 + i], sc, l_sum);
            } else {
                scale_smem[i] = 0.0f;
            }
        }
        m_global = m_max; l_global = l_sum;
        lse_ptr[t*16 + h] = (m_max == -INFINITY || l_sum <= 0.0f)
            ? -INFINITY
            : fmaf(m_max, 1.4426950408889634f, __log2f(l_sum));
    }
    __syncthreads();

    int d_idx = lane * 4;
    if (d_idx < 512) {
        float out0=0.f, out1=0.f, out2=0.f, out3=0.f;
        if (m_global != -INFINITY && l_global > 0.0f) {
            float inv_l = __fdividef(1.0f, l_global);
            #pragma unroll 16
            for (int s=0; s<64; ++s) {
                float coeff = scale_smem[s] * inv_l;
                if (coeff > 0.0f) {
                    int base = t*64*16*512 + s*16*512 + h*512 + d_idx;
                    float4 val = *reinterpret_cast<const float4*>(&O_tmp[base]);
                    out0 = fmaf(val.x, coeff, out0);
                    out1 = fmaf(val.y, coeff, out1);
                    out2 = fmaf(val.z, coeff, out2);
                    out3 = fmaf(val.w, coeff, out3);
                }
            }
        }
        __nv_bfloat162 b0 = __floats2bfloat162_rn(out0, out1);
        __nv_bfloat162 b1 = __floats2bfloat162_rn(out2, out3);
        uint32_t v0 = *reinterpret_cast<uint32_t*>(&b0);
        uint32_t v1 = *reinterpret_cast<uint32_t*>(&b1);
        uint2 ov = make_uint2(v0, v1);
        *reinterpret_cast<uint2*>(&output_ptr[t*16*512 + h*512 + d_idx]) = ov;
    }
}
